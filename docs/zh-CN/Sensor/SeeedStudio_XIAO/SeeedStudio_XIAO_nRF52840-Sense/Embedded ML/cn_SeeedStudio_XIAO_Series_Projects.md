# Seeed Studio XIAO 展览

XIAO系列开发板，其中“XIAO”代表小，是Seeed Studio推出的指甲盖大小的微型开发板。

我们在这里对这一系列开发板的应用进行清晰易懂的介绍，希望能为大家以后的使用提供参考和帮助。此外，我们还收集了使用这些小型开发板创作的小杰作，并热切期待您的 [上传](https://wiki.seeedstudio.com/Contribution-Guide/)。
